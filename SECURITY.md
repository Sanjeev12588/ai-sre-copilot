# Security Policy

## Supported Versions

We actively support and patch the latest release of AI SRE Copilot:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of AI SRE Copilot seriously. If you find any vulnerabilities or security concerns (such as prompt injection vectors, authentication bypasses, rate limit bypasses, or data exposure issues), please do not report them via public GitHub issues.

Instead, please send a detailed report to **security@sre-copilot.internal** (or the project maintainers).

Please include:
- A description of the vulnerability.
- A proof of concept (PoC) or reproduction steps.
- The potential impact.

We will review your submission and respond within 48 hours to coordinate a disclosure plan and deploy a patch.
